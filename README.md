"""Финальный проект спринта №13: 'CRUD для Yatube'."""  Результат представляет собой API для всех моделей приложения.

Для того, чтобы развернуть репозиторий локально и воспользоваться возможностями API, необходимо:
1.Открыть Github.
2.Нажать кнопку 'Fork' для переноса репозитория в свой аккаунт.
3.Открыть IDE.
4.Ввести команду 'git clone <ссылка на репозиторий из GitHub>' в терминал, чтобы клонировать репозиторий на свой компьютер.
5.Развернуть и активировать виртуальное окружение.
6.Установить необходимые зависимости из файла requirements.txt
с помощью команды 'pip install -r requirements.txt
7. Список доступных эндпоинтов:
    7.1 api/v1/api-token-auth/ (POST): передаём логин и пароль, получаем токен.
    7.2 api/v1/posts/ (GET, POST): получаем список всех постов или создаём новый пост.
    7.3 api/v1/posts/{post_id}/ (GET, PUT, PATCH, DELETE): получаем, редактируем или удаляем пост с идентификатором{post_id}.
    7.4 api/v1/groups/ (GET): получаем список всех групп.
    7.5 api/v1/groups/{group_id}/ (GET): получаем информацию о группе с идентификатором {group_id}.
    7.6 api/v1/posts/{post_id}/comments/(GET): получаем список всех комментариев поста с идентификатором post_id
        (POST): создаём новый комментарий для поста с идентификатором {post_id}.
    7.7 api/v1/posts/{post_id}/comments/{comment_id}/ (GET, PUT, PATCH, DELETE): получаем, редактируем или удаляем комментарий
        с идентификатором {comment_id} в посте с id=post_id.
8. Проверить работy API можно с помощью программы Postman.
